While True: note()
While True: note() - это программное обеспечение для ПК,
работающих на Windows OS, позволяющее быстро создавать и хранить заметки.

Приложение обладает функциями добавления новой заметки, конвертирования файла
типа txt в заметку и наоборот, удаления одной или нескольких заметок сразу,
получения информации о количестве строк в текущей заметке.

Для реализации своих возможностей "note ()" использует такие python библиотеки,
как sys, sqlite3, PyQT5.

Основное окно программы реализовано в виде класса NoteWidget,
макет для которого создан с помощью приложения QtDesigner и хранится
в одной директории вместе с основным кодом в виде файла формата ".py".
В коде присутствует второй класс - RemoveNotesWidget, создающий виджет,
позволяющий удалить несколько имеющихся заметок.

Общение программы с пользователем осуществляется с помощью диалоговых окон и окон сообщений.

Новые записи в "note()" вместе с их названиями хранятся в базе данных notes_information.sqlite.

Методы классов NoteWidget и RemoveNotesWidget часто обращаются к базе
с помощью языка запросов SQLite и получают необходимые данные о заметках, удаляют старые и добавляют новые.
В программе присутствует так же минимальная стилизация интерфейса - имеется
возможность изменять в некоторых пределах размер текста интерфейса и заметки.
При добавлении и удалении заметок, изменяется счётчик их количества, показывающий,
сколько пользователем было создано заметок.

При обновлении окна information появляется возможность узнать количество строк в открытой заметке.
